# ASISA для ВНЖ / визы в Испании (ключ для приезжих)

Самый частый запрос аудитории Seguro Tenerife — медстраховка, которая **подходит
для оформления ВНЖ/визы** и даёт **сертификат** для консульства / иммиграционных
органов. У ASISA это линия «для иностранцев» (`salud_extranjeros`).

## Что предлагать
| Кейс клиента | Продукт | id |
|---|---|---|
| Переезд на ВНЖ (non-lucrativo, работа, семья), продление NIE/TIE | **ASISA Health Residents** | `extranjeros-residents` |
| Студент/преподаватель, виза на учёбу (>90 дней) | **ASISA Health Students** | `extranjeros-students` |
| Общая страница линии для иностранцев | Seguro para extranjeros | `extranjeros-general` |

## Почему именно эти
- **Без доплат (copagos) и без периодов ожидания (carencias)** — требование для визы.
- **Моментальный PDF-сертификат**, принимаемый консульствами и иммиграцией.
- Полное покрытие: первичная помощь, специалисты, скорая 24ч, госпитализация и
  хирургия, диагностика, протезы/импланты/трансплантация (Residents), телемедицина,
  ассистанс в поездках вне Испании 25 000 €, репатриация.

## Что важно сказать клиенту
- **Residents**: можно оформить **за 150 дней до приезда**; первый год — единый
  срок, со 2-го года нужен испанский IBAN; не возвращается, кроме официального
  отказа в визе.
- **Students**: срок **2–12 месяцев**, договор разовый (не продлевается), оплата
  только картой.
- Туристическая страховка (`viaje-*`) подходит для **Schengen краткосрочно**, но
  **НЕ для ВНЖ** — для резиденции нужен медполис из линии для иностранцев.

## Цена
Не опубликована — считается индивидуально (тарификатор asisa.es / телефон). Не
называть цифры «из головы»: при интересе передавать менеджеру.

Источник: <https://www.asisa.es/seguros-medicos/extranjeros> и подстраницы
`/residencia`, `/estudios`. Детали по каждому продукту — в `catalog.json`.
