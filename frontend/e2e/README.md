# E2E (Playwright) — Волна 4

E2E-тесты в реальном браузере против собранных SPA. Backend-API подменяется
детерминированными стабами (`page.route`), поэтому тесты не зависят от БД/сети и
проверяют именно UI-флоу. Живой API покрыт backend-smoke и k6-нагрузкой.

## Запуск
```bash
cd frontend
pnpm install
pnpm exec playwright install chromium   # один раз — скачать браузер
pnpm e2e                                 # сборка web/admin + preview + тесты
pnpm e2e:ui                              # интерактивный режим
```

Playwright сам поднимает два preview-сервера (web :4173, admin :4174), ждёт
готовности и гоняет тесты.

## Покрытие
- `tests/web-chat.spec.ts` — проход чат-подбора до экрана хендоффа; смена языка
  перерисовывает чат без перезагрузки.
- `tests/admin-auth.spec.ts` — нет сессии → форма входа; неверный пароль → ошибка;
  верный пароль → дашборд со списком лидов.
